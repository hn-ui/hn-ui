# IconGroup


## 需求说明

      自定义icon组：
      
        1. 定义组内的icon是否有边框；
        
        2. 定义组内的icon是否最多只能选中一个；

## Attributes

| 参数   | 说明                 | 类型    | 可选值 | 默认值 |
| ------ | -------------------- | ------- | ------ | ------ |
| border | 组内icon是否设置边框 | boolean | -      | false  |
