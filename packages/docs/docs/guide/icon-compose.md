# IconCompose

## Requirement

      icon编排组件：
      
        1. 将两个aqsc-icon放置在该组件内，第二个组件以“child”的样式显示在第一个组件的右下方；
        2. 有且只有两个aqsc-icon放置在该组件内，才会生效
